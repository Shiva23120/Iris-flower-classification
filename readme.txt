A Flask application with integrated ML model which classifies the folower-species as iris-setosa or 
iris-versicolor or iris-virginica based on the featues sepal length,sepal width,petal length and 
petal width.The model is trained using LogisticRegression algorithm
This output is based on the input which is taken in form by user.

Requirements:


pip install numpy

pip install pandas

pip install sklearn

pip install flask

pip install pickle


pip install seaborn
